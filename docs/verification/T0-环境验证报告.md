### 3.1 环境配置问题

直接执行 Hvigor 时出现：

```text
00303217 Configuration Error
Error Message: Invalid value of 'DEVECO_SDK_HOME' in the system environment path.
```

根因是当前命令行进程没有有效的 `DEVECO_SDK_HOME`。将该变量仅在当前 PowerShell 进程中设置为 DevEco SDK 根目录后，任务列表、构建与测试都成功：

```powershell
$env:DEVECO_SDK_HOME='D:\Dev\IDE\DevEco Studio\sdk'
```

注意：变量值必须是 `...\sdk`，不是 `...\sdk\default`；后者会触发 `00303168 SDK component missing`。本次没有擅自写入用户或系统级环境变量。开始后续任务前应持久化正确变量，或在项目开发命令说明中统一设置。

1. API 26、Stage、ArkTS、OHPM、Hvigor 和本地测试链路可继续使用，无需重建工程。
2. 后续命令应统一设置正确的 `DEVECO_SDK_HOME`；当前会话临时配置已可稳定构建。
